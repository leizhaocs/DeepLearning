Folder Structure

The source code of DeepLearning framework contains the following folders and files:

|-- LICENSE                       License file.
|-- Makefile                      Makefile for make.
|-- README.md                     Compile and run information.
|-- cfg/                          Contains config files.
|-- data/                         Contains training datasets.
    |-- mnist/                    The MNIST dataset.
        |-- download_mnist.py     Run this python code to automatically download MNIST dataset.
    |-- cifar10/                  The Cifar10 dataset.
        |-- download_cifar10.py   Run this python code to automatically download cifar10 dataset.
    |-- svhn/                     The SVHN dataset.
        |-- download_svhn.py      Run this python code to automatically download SVHN dataset.
|-- obj/                          This folder contains all object files generated during the compilation.
|-- python/                       Python script
|-- weights/                      The saved weights after training are stored in this folder by default.
|-- src/                          This folder contains all the source code of the framework.
